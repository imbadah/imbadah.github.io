عن PdfSwift

PdfSwift هو مشروع مفتوح المصدر يهدف إلى توفير أدوات PDF بسيطة وقوية تعمل مباشرة داخل المتصفح بدون الحاجة إلى رفع الملفات إلى سيرفرات خارجية.

الأهداف الرئيسية

  • الحفاظ على خصوصية المستخدمين
  • سرعة المعالجة (client-side فقط)
  • دعم اللغة العربية بشكل كامل (RTL)
  • تصميم نظيف وعصري
  • إمكانية التوسع بأدوات إضافية مستقبلاً

التقنيات المستخدمة

HTML + CSS + JavaScript نقي، مع مكتبة pdf-lib لمعالجة الملفات PDF.
